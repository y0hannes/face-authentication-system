# FaceAuth-AI

## Overview
FaceAuth is an AI-powered face login system. Users can register using their face and log in without passwords. This repository contains the ML pipeline and UI placeholders.

## Current Status
- ML pipeline is set up with dummy data
- Feature engineering, training, and evaluation scripts work
- Trained model saved as `models/face_model.pkl`
- Ready for real image data integration
